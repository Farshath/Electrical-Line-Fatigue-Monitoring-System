# Electrical Line Fatigue Monitoring System

An IoT-based smart monitoring system using Arduino Uno and NodeMCU ESP8266 to detect overheating and overloading conditions in electrical lines in real time.

---

## Features

- Real-time temperature monitoring
- Current overload detection
- SAFE / FATIGUE / DANGER classification
- LCD live display
- LED and buzzer alerts
- Blynk IoT cloud integration
- Smartphone monitoring

---

## Technologies Used

- Arduino Uno
- NodeMCU ESP8266
- Embedded C
- Blynk IoT
- IoT Sensors

---

## Components Used

- LM35 Temperature Sensor
- Potentiometer
- 16x2 LCD Display
- LEDs
- Active Buzzer
- Arduino Uno
- NodeMCU ESP8266
